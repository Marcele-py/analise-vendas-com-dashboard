Análise de Vendas Anual

Projeto de análise de vendas de um ano completo utilizando Excel. A base de dados contém informações de data, produto, categoria, valores, região, vendedor e formas de pagamento.

Foram utilizadas funções como PROCV, fórmulas básicas, tabelas dinâmicas, gráficos e um dashboard dinâmico com filtros por região e categoria. O projeto apresenta análises de receita total, receita mensal, vendas por categoria, forma de pagamento e desempenho por vendedor.
Todos os dados utilizados nessa planilha são fictícios, não possui quaisquer dados sensíveis.
